# 📊 Interactive Sales Analysis Dashboard (Excel)

## 🎯 Project Overview
This project was developed as part of my professional training in the **Digital Pioneers (Digilians)** initiative. The goal was to transform raw sales data into an interactive visual dashboard to help business owners make data-driven decisions.

## 🛠️ Key Features & Tools
- **Advanced Excel:** Used Pivot Tables, VLOOKUP, and complex formulas for data cleaning.
- **Interactive Dashboard:** Built with **Slicers** and **Timelines** to filter data by region, category, or date.
- **Data Visualization:** Created dynamic charts (Bar, Line, and Pie charts) to represent sales trends and KPIs.
- **Insights:** Analyzed year-over-year growth and identified top-performing products.
