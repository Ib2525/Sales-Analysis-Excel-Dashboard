## 🎯 Project Overview
This project is an end-to-end interactive Excel dashboard designed to analyze sales performance, profitability, and regional trends for a retail business. Developed as part of the **MCIT Digital Pioneers (Digilians)** program, it demonstrates advanced data manipulation, modeling, and visualization techniques.

## 📈 Key Performance Indicators (KPIs) Tracked
- **Total Sales:** $241,131 💰
- **Total Profit:** $18,746 📈
- **Total Orders:** 1,000 🛍️
- **Total Margin:** 8% 📊

## 🛠️ Technical Skills & Tools Used
- **Data Transformation:** Advanced Formulas & Data structuring.
- **Data Modeling:** Interconnected Pivot Tables with dynamic data ranges.
- **Interactive UI/UX:** Custom Navigation Menu (Report 1, 2, 3), Slicers, and Timelines.
- **Visualizations:** Geographic Maps, Donut charts, Bar/Column charts, and Trendlines.

## 💡 Key Business Insights
1. **Regional Dominance:** The West region leads in sales (29%), followed closely by the East (28%).
2. **Shipping Preferences:** Standard Class shipping is the most utilized and generates the highest overall profit.
3. **Product Performance:** Phones and Chairs are the top-selling sub-categories, driving the majority of revenue, while Copiers show significant growth potential.
